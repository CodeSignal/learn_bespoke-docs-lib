Docs Lib Help

Viewing Documents

Select a document from the sidebar to view its contents. The document is displayed in the main viewer area with a comment panel on the right.

Editing Content

Double-click any section (heading, paragraph, list item, or table cell) to enter edit mode. While editing:

  • A blue outline highlights the section being edited
  • Type to modify the text directly
  • Click outside the section or press Enter (for headings/table cells) to save
  • Press Escape to cancel without saving

A small pencil icon appears when you hover over a section, indicating it can be edited.

Commenting

Single-click any section to open the comment panel on the right side. The panel displays existing comments and a form to add new ones.

  • Type your comment and click Post to add it
  • Sections with existing comments show a blue left border and a comment count badge
  • Click the same section again to close the comment panel
  • The comment panel stays aligned with the active section as you scroll

Data Storage

All edits and comments are saved in your browser's local storage and persist across page refreshes.
